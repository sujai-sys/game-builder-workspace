Prompt Guide

How to ask the AI for a game

  1. Click Open OpenAI or Open Gemini.
  2. Paste a prompt like the example below.
  3. Ask the AI to return one complete HTML file with CSS and JavaScript included.
  4. Copy the generated code.
  5. Come back here and paste it into Programming Render.
  6. Click Render to execute and play the game.
