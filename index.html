ChatKit Frontend Ready

How to finish ChatKit

  1. Create an OpenAI workflow in Agent Builder and copy the workflow ID.
  2. Create a backend endpoint at /api/chatkit/session that calls POST /v1/chatkit/sessions.
  3. Return the client_secret from that backend to this site.
  4. Point the saved endpoint above to that backend URL.

Your frontend layout is ready now. The missing piece is the backend session creator.

  • Workflow backend: OpenAI Agent Builder workflow
  • Session endpoint needed: POST /api/chatkit/session
  • Unique user id required by OpenAI for each user session
